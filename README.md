Program „Książka serwisowa pojazdu” w języku Java w oparciu środowisko graficzne JavaFX, implementujący bazę danych MySQL z zastosowaniem logowania użytkowników wraz z kontrolą poziomów dostępu. Założeniem programu była praktyka w wykorzystaniu języka Java z zachowaniem standardów JavaBeans, wykorzystanie bazy danych MySQL w programie Java, stworzenie interfejsu graficznego JavaFX przy wykorzystaniu SceneBuilder.
